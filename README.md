## How to run the programs

#### Activate venv windows
````.\env\Scripts\activate.bat````

#### Use 'keygen.py'
```python keygen.py ./pub.txt ./priv.txt 1024```
#### Use 'encrypt_file.py'
``` python encrypt_file.py fixe.txt .\pub.txt encrypted.bin```
#### Use 'decrypt_file.py'
``` python decrypt_file.py encrypted.bin .\priv.txt decrypted_output.txt ```.
